# Contributing to NGO Career

Thank you for your interest in contributing to NGO Career! We welcome contributions from everyone.

## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Create a new branch** for your feature or bug fix:
    ```bash
    git checkout -b my-feature-branch
    ```
4.  **Make your changes** and commit them with clear messages.
5.  **Push your branch** to your fork on GitHub.
6.  **Submit a Pull Request** to the main repository.

## Reporting Bugs

If you find a bug, please create a new issue using the [Bug Report Template](.github/ISSUE_TEMPLATE/bug_report.md). Include as much detail as possible to help us reproduce and fix the issue.

## Feature Requests

If you have an idea for a new feature, please submit an issue using the [Feature Request Template](.github/ISSUE_TEMPLATE/feature_request.md).

## Code Style

- Please follow the existing code style and conventions.
- Ensure your code is clean and well-documented.
