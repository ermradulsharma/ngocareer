# Security Policy

## Supported Versions

Please check the project releases to find the latest supported versions.

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please do NOT open a public issue.

Instead, please report it via email to the maintainers/project owners. We will acknowledge your report and keep you updated on the progress of the fix.

Please include:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Any relevant logs or output.
